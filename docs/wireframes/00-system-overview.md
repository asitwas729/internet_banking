## 사이트맵 (라우트 트리)

```
(personal) ─ 대고객
├─ /login · /login/pin · /logout            인증
├─ /dashboard                               메인 대시보드
├─ /inquiry/accounts · /inquiry/transactions 조회
├─ /accounts/[accountId]                    계좌 상세
├─ /products/deposit[/...]                  예적금 (목록·상세·가입·해지)
├─ /products/loan[/...]                     대출 (상품·가심사·관리·현황)
├─ /loans/apply[/...]                       대출 신청 멀티스텝
├─ /transfer[/...]                          이체 (당행·타행·자동·결과)
├─ /cert · /cert-biz                        인증서 발급/관리
├─ /banking/... · /settings · /mypage       뱅킹설정·내정보
├─ /support/...                             고객지원 (상담·FAQ·공지)
├─ /biz/...                                 법인뱅킹
└─ /other-bank/...                          오픈뱅킹

(admin) ─ 직원/백오피스
├─ /admin/login · /admin/dashboard
├─ /admin/members[/id] · /admin/member-status   직원관리
├─ /admin/customers · /admin/audit-log · /admin/audit/quarantine
├─ /admin/loan/review[/...] · /documents · /contracts · /collateral
│        · /credit-report · /identity · /rate-policy · /calendar
│        · /eod · /auto-review-sim · /break-glass · /audit
├─ /admin/ai/doc-review · /rag-documents · /advisory-rules · /audit-risk
├─ /admin/consultation/customer
└─ /admin/join-stats
```

```
┌──────────────────────────────────────────────────────────────────┐

├──────────────────────────────────────────────────────────────────┤

├──────────┬───────────────────────────────────────────────────────┤

```

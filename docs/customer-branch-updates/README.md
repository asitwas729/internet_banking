## 2026-05-28: 가입 적금 계좌조회 표시 보정

- 예금/적금 가입 완료 버튼 문구를 `가입완료`로 변경했습니다.
- 적금 가입 시 가입 금액을 검증하고 가입 계좌 타입을 `적금`으로 저장하도록 보정했습니다.
- 계좌조회 화면에서 `joinedAccounts`에 저장된 가입 계좌 타입을 상품명 기준으로 보정해, 가입한 적금이 적금 섹션에 표시되도록 했습니다.
- 기존 화면 구조는 유지하고, 브라우저 임시 저장(localStorage) 기준의 가입 계좌 표시만 수정했습니다.

## 확인 방법

1. 프론트엔드를 실행합니다.
   ```powershell
   cd web
   npm run dev
   ```
2. 같은 포트에서 로그인 후 적금 상품을 가입합니다.
3. `/inquiry/accounts`로 이동해 예금 탭 안의 적금 섹션에 가입한 적금 계좌가 표시되는지 확인합니다.

## 참고

- 현재 프론트 가입 완료 처리는 실제 DB 저장이 아니라 브라우저 `localStorage`의 `joinedAccounts`에 저장하는 방식입니다.
- 포트가 바뀌면 브라우저 저장소도 달라지므로, 가입과 조회는 같은 포트에서 확인해야 합니다.

# 🖥 LON 여신계 화면 기능 정의서

본 문서는 여신계(Loan) 도메인의 화면(대고객 11 + 백오피스 11) 별
**목적 + 주요 기능**만 정의한다. API 규격·상태머신·JSON 예시 등은 범위 밖이다.

- 기반 ERD: [loan_erd.md](./loan_erd.md)
- API 명세는 Notion DB 의 LON 화면 기능 표 참조

---

## 1. 대고객 트랙

### A-1. 대출상품 조회

**목적**: 고객이 가입 가능한 대출 상품을 둘러보고 비교한다.

**주요 기능**:
- 판매 중(`prod_status_cd=ACTIVE`) 상품 목록 조회
- 대출 유형(신용/주담대/전세 등) 필터, 정렬 (금리/한도/기간)
- 상품 단건 상세 (기본금리, 한도 범위, 기간 범위, 담보·보증 필수 여부, 상품설명서 PDF 링크)
- 우대금리 정책 목록 표시 (급여이체·자동이체·카드실적 등)
- "한도조회로 이동" CTA

---

### A-2. 한도조회 (가심사)

**목적**: 본 신청 전 예상 한도·금리를 즉시 확인한다.

**주요 기능**:
- 신청 정보 입력 (희망 금액·기간·자금용도·추정 연소득·직업유형)
- 신용조회 동의 (1회성, 가심사 한정)
- 가심사 실행 → 예상 한도·예상 금리·추정 신용등급/점수 표시
- 거절 시 사유 안내
- "대출 신청으로 이어가기" CTA (가심사 결과를 신청에 prefill)
- 결과 저장 후 마이페이지에서 재조회 가능

---

### A-3. 대출 신청

**목적**: 본 신청서를 작성·제출하여 심사 프로세스에 진입한다.

**주요 기능**:
- 신청 정보 입력/수정 (희망 금액·기간·자금용도·상환방식)
- 본인확인(IDV) — PASS/카카오/공동인증 등
- 신용조회 동의 (본심사용 — 가심사 동의와 별도)
- 신청서류 업로드 (소득증빙, 재직증빙 등) + OCR 진행 상태 표시
- 담보 정보 입력 (담보 필수 상품인 경우) + 감정평가 신청
- 보증인 추가 + 보증 동의 링크 발송 (보증 필수 상품)
- 임시저장 → 이어쓰기 지원
- 청약 철회 안내 (금소법 14일) 사전 고지
- 신청 완료 → 신청번호 발급, A-4 로 이동

> 모바일/웹 공용. 멱등성 키로 중복 신청 차단.

---

### A-4. 신청 진행 현황

**목적**: 고객이 자기 신청의 단계별 진행 상태를 한 화면에서 확인한다.

**주요 기능**:
- 신청 단계 타임라인 (신청 → IDV → 서류검증 → 가심사/CB → 본심사 → 승인/거절 → 약정대기 → 실행)
- 각 단계별 상태·소요시간·다음 액션
- 추가 서류 요청 알림 + 업로드 진입
- 재심사 요청 (거절 시)
- 신청 취소 (고객 직접)
- 거절 시 사유 표시 + 재신청 안내
- 승인 완료 시 "약정 체결로 이동" CTA

---

### A-5. 약정 체결

**목적**: 승인된 대출 조건을 확인하고 약정서를 전자서명한다.

**주요 기능**:
- 계약 사전안내 화면 (승인 한도·금리·기간·상환방식·총 상환 예상액)
- 약정서 PDF 미리보기
- 약관·동의 일괄 체크 (필수/선택 분리)
- 전자서명 (본인확인 1회 더)
- 약정서 PDF 다운로드 / 메일 발송
- 보증보험 가입 안내 (해당 상품)
- 보증인 동의 진행 현황 (보증 상품)
- 약정 완료 → A-6 로 이동

---

### A-6. 대출 실행

**목적**: 약정된 대출 금액을 지정 계좌로 실행(입금)한다.

**주요 기능**:
- 약정 정보 요약 표시
- 지급 계좌 확인/변경 (예금주 본인 검증)
- 실행 수수료·인지세 안내
- 실행 요청 (멱등성 키 포함)
- 실행 결과 화면 (실행 금액, 실행 일시, 이자 기산일)
- 같은 영업일 내 실행 취소 진입
- 상환계좌 등록 / 자동이체 신청 유도
- 가상계좌 발급 안내

---

### A-7. 내 대출 현황

**목적**: 고객이 보유한 대출 계약 목록과 잔액을 한눈에 본다.

**주요 기능**:
- 활성 계약 목록 (계약번호·상품명·약정금액·잔액·다음 상환일·다음 상환액)
- 계약 단건 상세 (적용금리 구성: 기준+가산-우대, 약정기간, 상환방식)
- 종결된 계약 분리 표시
- 거래원장(실행·상환·이자) 통합 조회 진입
- 증명서 발급 진입 (A-11)
- 상환 스케줄 진입 (A-8)
- 상환 처리 진입 (A-9)

---

### A-8. 상환 스케줄

**목적**: 회차별 상환 계획과 잔액 추이를 확인한다.

**주요 기능**:
- 회차별 표시 (회차·예정일·원금·이자·합계·상환후잔액)
- 적용금리 변경 이력에 따라 스케줄 버전 분기 표시
- 납부 완료/예정/연체 회차 구분
- 영업일 보정된 실제 납부일 표시
- 다음 회차 자동이체 출금일 안내
- 사전 안내장 발송 이력 표시

---

### A-9. 상환 처리

**목적**: 고객이 즉시 상환·중도상환을 실행한다.

**주요 기능**:
- 약정 회차 상환 (예정 금액 그대로)
- 추가 상환 / 중도 상환 입력 (부분/전액)
- 중도상환수수료 사전 계산 표시
- 상환 채널 선택 (출금계좌, 가상계좌 입금)
- 상환 결과 영수증 (원금/이자/연체이자/수수료 분배)
- 자동이체 등록/계좌변경/해지
- 자동이체 실패 시 재시도 안내

---

### A-10. 연체 현황

**목적**: 연체 발생 시 사실과 해소 방법을 안내한다.

**주요 기능**:
- 연체 시작일·연체일수·연체 원금/이자·연체 가산금리 표시
- 연체 단계(STAGE_1/2/…) 와 다음 단계 도래 시점 안내
- 즉시 해소 결제 진입 (A-9 와 연계)
- 알림·독촉장 발송 이력 표시
- 신용정보 등록 예정/완료 고지

---

### A-11. 증명서 발급

**목적**: 대출잔액·부채·상환 등 각종 증명서를 발급받는다.

**주요 기능**:
- 증명서 유형 선택 (대출잔액증명서, 부채증명서, 상환증명서 등)
- 용도 입력 (제출처)
- 발급 채널 선택 (모바일/이메일/PDF)
- 발급 완료 → 증명서 PDF 다운로드 / 발급번호 표시
- 발급 이력 조회 (재발급 포함)

---

## 2. 백오피스 트랙

> 모든 백오피스 화면은 **관리자 권한 필요**. 결재·승인 행위는 결재 라인 검증을 거친다.

### B-1. 대출상품 관리

**목적**: 상품 라이프사이클(등록·수정·단종) 과 우대금리 정책을 관리한다.

**주요 기능**:
- 상품 목록 (상태 필터: DRAFT/ACTIVE/DISCONTINUED)
- 상품 등록 (기본금리·한도·기간·필수옵션·상품설명서 URL)
- 상품 수정 (금리·한도 변경 시 변경이력 기록)
- 상품 단종 처리 (판매 종료일, 사유)
- 우대금리 정책 CRUD (조건·적용 bps·중복 한도)
- 상품별 변경 이력 조회

---

### B-2. 신청 접수 (창구)

**목적**: 영업점 직원이 고객 신청을 대행 접수·보정한다.

**주요 기능**:
- 고객 검색 (CI/주민번호/계좌)
- 신청서 작성 (대고객 A-3 와 동일 입력 + 채널 코드 = 창구)
- 서류 스캔 업로드 / OCR 결과 검증·수동 보정
- 본인확인 대행 (창구 신분증 확인 기록)
- 임시저장 / 제출
- 동일 고객의 진행 중 신청 조회

---

### B-3. 심사 워크리스트

**목적**: 심사자가 본인에게 라우팅된 건을 한 화면에서 처리한다.

**주요 기능**:
- 내 워크리스트 (배정 건수·SLA 임박 표시)
- 필터 (심사 유형, 신청 채널, 금액 구간, 자동/수동)
- 우선순위 정렬 (마감 임박, 고액, VIP)
- 일괄 라우팅 / 재배정
- 건당 클릭 → B-4 진입

---

### B-4. 심사 상세 / 결재

**목적**: 단건 신청의 모든 정보를 보고 승인/거절·결재 라인을 처리한다.

**주요 기능**:
- 신청 요약 (희망·자금용도·소득·직업)
- CB 평가 결과 (등급·점수·PD·결정)
- LTV / DSR 산출 결과
- 담보·보증·서류 점검 결과
- 점검 항목 체크리스트 입력
- 승인 입력 (금액·금리·기간) / 거절 입력 (사유·비고)
- 재심사 결과 반영
- 결재 라인 상신·승인 (다단계 결재)
- 처리 후 워크리스트로 복귀

---

### B-5. 약정 관리

**목적**: 승인 건의 약정 체결 진행과 사후 변경을 관리한다.

**주요 기능**:
- 약정 대기/체결 건 조회
- 계약 사전안내 발송 처리
- 약정서 재발급 / PDF 출력
- 보증보험 등록·해지
- 계약 조건 변경 (상환방식, 거치기간 등)
- 만기 연장 처리
- 한도 갱신 신청·승인 (한도형 상품)
- 금리 변경 처리 (수동) + 스케줄 재생성

---

### B-6. 실행 처리

**목적**: 대출 실행 거래를 모니터링하고 운영 이슈를 처리한다.

**주요 기능**:
- 실행 대기/완료 건 조회
- 실행 (창구 대행)
- 분할실행 / 추가실행 (한도형)
- 같은 영업일 실행 취소
- 회계 전표번호 매핑 조회
- 실행 수수료·인지세 등록

---

### B-7. 상환 처리

**목적**: 자동이체·수동 입금·역분개 등 모든 상환 운영을 처리한다.

**주요 기능**:
- 자동이체 배치 실행/재시도 모니터링
- 자동이체 결과 콜백 확인
- 수동 입금 처리 (창구)
- 가상계좌 입금 매칭 보정
- 역분개 (운영 오류 보정)
- 수수료 환급 / 면제 처리
- 상환 사전안내장 발송
- 일별 이자 발생 배치 결과 조회

---

### B-8. 연체 관리

**목적**: 연체 발생 건을 추적하고 알림·독촉을 진행한다.

**주요 기능**:
- 연체 건 목록 (단계·일수·금액·고객)
- 연체 일별 스냅샷 조회
- 연체 알림 발송 / 독촉장 발송
- 단계 전이(STAGE_1→2→…) 결과 확인
- 해소 처리 (전액 상환 후 자동 해소)
- 고객별 연체 이력 추적

---

### B-9. 신용정보 신고 관리

**목적**: KCB/NICE 등 신용정보기관 신고 전문을 관리한다.

**주요 기능**:
- 신고 대기/전송/접수확인 건 조회
- 신고 유형별 필터 (신규대출/연체/해소)
- 신고 전문(JSON) 미리보기
- 수동 재전송 / 외부거래번호 추적
- 접수확인(ACK) 도착 모니터링
- 실패 건 사유 확인 + 재시도

---

### B-10. 약정 종료 처리

**목적**: 완납·중도해지·기한이익상실 등 계약 종결을 처리한다.

**주요 기능**:
- 종결 대상 건 조회 (만기 도래, 완납, 사고)
- 종결 유형 선택 (정상완납/중도상환/기한이익상실)
- 최종 정산 계산 (원금·이자·수수료·중도상환수수료)
- 종결 증서 PDF 발급
- 담보 해제 / 보증 해제 동시 처리
- 종결 후 신용정보 신고 트리거

---

### B-11. 대시보드 / 통계

**목적**: 여신 운영 KPI 와 NPL 지표를 한 화면에서 모니터링한다.

**주요 기능**:
- NPL 비율·총액·단계별 분포
- 신규 신청·실행 추이 (일/주/월)
- 평균 심사 소요시간, 자동/수동 비율
- 연체율·자동이체 성공률
- 차주별 통합 익스포져 검색
- 관리자 대출 검색 (B-2~B-10 진입점)
- 기간별 추출 (Excel 다운로드)

# 대출심사 편향 탐지 통계 기준서 v1.0

## 1. 개요

본 기준서는 미국 주택담보대출공시법(HMDA, Home Mortgage Disclosure Act) 분석 방법론을 한국 대출심사 환경에 적용하여, 심사관 수준의 집단별 편향 여부를 통계적으로 판별하기 위한 기준을 정의한다.

## 2. 핵심 지표 정의

### 2.1 불균등 영향 비율 (Disparate Impact Ratio, DIR)

DIR = 보호 집단 승인율 / 기준 집단 승인율

- DIR < 0.80: 불균등 영향 의심 (4/5 Rule, EEOC 기준)
- DIR < 0.70: 불균등 영향 강한 의심
- DIR < 0.60: 명백한 편향 신호 (즉각 조사 필요)
- DIR 0.80 ~ 1.20: 허용 범위 (중립 판정)

보호 집단 예시: 여성, 외국인, 비정규직, 고령자(65세 이상), 저소득층

### 2.2 표준편차 기반 편차 탐지 (Sigma Rule)

개인 심사관 승인율과 동일 코호트 내 동료 평균 승인율의 편차를 표준화:

Z = (개인 승인율 - 코호트 평균 승인율) / 코호트 표준편차

- |Z| > 2.0σ: WARNING 수준 편향 신호
- |Z| > 3.0σ: CRITICAL 수준 편향 신호 (즉각 감사 필요)
- |Z| ≤ 2.0σ: 정상 범위

### 2.3 거절률 편차 (Adverse Action Rate Disparity)

거절률 편차 = |개인 거절률 - 코호트 평균 거절률|

- 편차 > 10%p: 경미한 이상
- 편차 > 20%p: 중간 이상 (WARN)
- 편차 > 30%p: 심각 이상 (CRITICAL)

## 3. 통계적 유의성 검정

소표본 편향 탐지 시 Fisher's Exact Test 또는 Chi-square 검정 적용:

- p-value < 0.05: 통계적 유의 (편향 의심 근거 성립)
- p-value < 0.01: 강한 통계적 유의 (편향 판정 근거 성립)
- 표본 크기 < 30: 통계 결과 신뢰도 낮음, 추가 기간 확보 후 재평가

## 4. 코호트 구성 기준

동등한 비교를 위해 아래 조건을 통제한 코호트를 구성한다:

- 상품 유형 (신용대출, 담보대출, 전세자금대출 등)
- 신청 기간 (동일 분기 내)
- 지역 (동일 영업점 또는 권역)
- DSR 구간 (< 40%, 40~60%, > 60%)
- 신용등급 구간 (1~3등급, 4~6등급, 7등급 이하)

## 5. 복합 편향 탐지

단일 지표가 아닌 복수 지표의 동시 발현 시 편향 판정 신뢰도 증가:

- DIR 위반 + Z > 2σ: 복합 편향 신호 (높은 신뢰도)
- DIR 위반 + Z > 3σ: 복합 편향 판정 (매우 높은 신뢰도)
- 동일 심사관이 3개월 연속 편향 신호 발생: 구조적 편향 가능성

## 6. 측정 주기

- 일일 모니터링: 당일 심사 완료 건 실시간 집계
- 주간 리포트: 최근 5영업일 누적 기준 편향 지표 산출
- 월간 정기 감사: 전월 1개월 전체 데이터 기반 심층 분석
- 분기 검토: 코호트 재설정 및 임계값 재보정 (시장 변화 반영)
